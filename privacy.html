github-trending-ripos GitHub Trending on TikTok

Privacy Policy

App: github-trending-ripos  |  Last updated: 2026-07-10

github-trending-ripos is a personal automation tool that generates short videos summarizing trending GitHub repositories and publishes them to the operator's own TikTok account using TikTok's Content Posting API. Only one TikTok account — belonging to the bot's creator — is ever involved.

Information We Access

When the operator authorizes the app via TikTok Login Kit, the app receives:

  • The account's TikTok Open ID (an opaque, non-personally identifiable identifier)
  • The account's public display name and avatar — used only to confirm which account a video is about to be published to
  • An OAuth access token and refresh token scoped to video.publish, used solely to upload and publish videos to that same account

How We Use It

This information is used exclusively to authenticate API requests and to publish videos. It is stored locally on the operator's machine and is never sold, shared, or transmitted to any third party. No video content, account data, or tokens are sent anywhere other than TikTok's own API endpoints.

Data Retention

Access and refresh tokens are stored locally until the operator revokes access via TikTok's account settings or deletes them manually. No other personal data is retained by the app.

Cookies

github-trending-ripos does not use cookies. This website contains no tracking, analytics, or third-party scripts.

Data Security

Credentials are stored only on the operator's own machine and are not transmitted to any external server. The operator is responsible for maintaining the security of their local environment.

Your Rights

If you believe any data related to your account has been processed in error, you may contact us at the address below and we will respond within 30 days. You may also revoke this app's access to your TikTok account at any time through TikTok's security settings.

Third Parties

We do not share any data with third parties. The only external service contacted is TikTok's own API.
