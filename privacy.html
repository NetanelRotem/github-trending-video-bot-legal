Privacy Policy

App: GitHub Trending Video Bot  |  Last updated: 2026-07-06

GitHub Trending Video Bot is a personal, single-account automation tool that generates short videos summarizing trending GitHub repositories and publishes them to the operator's own TikTok account using TikTok's Content Posting API. It is not a multi-user service and does not act on behalf of any TikTok account other than the one its operator has explicitly authorized.

What information we access

When the operator connects their TikTok account via TikTok Login Kit, the app receives:

  • The account's TikTok Open ID (an opaque identifier)
  • The account's public display name and avatar (used only to confirm which account a video is about to be published to)
  • An OAuth access token and refresh token scoped to video.publish, used solely to upload and publish videos to that same account

What we do with it

This information is used only to authenticate API requests and to publish videos to the authorized account. It is stored locally on the operator's own machine and is never sold, shared, or transmitted to any third party. No video content, account data, or tokens are sent anywhere other than TikTok's own API endpoints.

Data retention

Access and refresh tokens are kept locally until the operator revokes access (via TikTok's account settings) or deletes them. No other personal data is retained.

Third parties

We do not share any data with third parties. The only external service contacted is TikTok's own API.
