🐹 What is LYLA?

LYLA is an automated content bot that discovers the most exciting open-source projects on GitHub every single day and turns them into short, engaging TikTok videos. From trending AI tools to developer utilities — LYLA finds them, narrates them, and posts them so the developer community never misses what's next.

⚡ How it works

  • Scrapes GitHub's Trending page daily for the hottest repositories
  • Generates a narrated short video for each project using AI voice synthesis
  • Publishes each video to TikTok using TikTok's Content Posting API
  • No user data is collected — only the operator's own account is used
