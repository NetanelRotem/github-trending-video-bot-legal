github-trending-ripos

Turn trending GitHub projects into short videos for your TikTok

📱 What is github-trending-ripos?

github-trending-ripos is a desktop app for developer-focused content creators. It gathers each day's trending open-source repositories from GitHub and helps you turn them into short, narrated vertical videos — from trending AI tools to developer utilities — that you review, approve, and post to your own TikTok account.

⚡ How it works

  • Browse the day's trending GitHub repositories inside the app
  • Pick a project and generate a short narrated video about it using AI voice synthesis
  • Preview the video and make any changes before it goes anywhere
  • Sign in with your TikTok account and approve the post — nothing is ever posted without your explicit confirmation

We only request basic profile info (display name and avatar) so you can see which TikTok account is connected. Videos are posted only to your own account, and only after you approve each one.
